## 起動コマンド

| コマンド       | 内容                         |
| -------------- | ---------------------------- |
| yarn storybook | storybook 起動               |
| yarn dev     | アプリ 起動 (localhost:3000) |

## 命名規則

| 対象                             | 規則                       |
| -------------------------------- | -------------------------- |
| コンポーネントのディレクトリ     | 先頭大文字、キャメルケース |
| コンポーネント定義ファイル       | index.tsx                  |
| ストーリー定義ファイル           | \*.stories.tsx           |
| プロパティインターフェイス名     | (コンポーネント名) + Props |
